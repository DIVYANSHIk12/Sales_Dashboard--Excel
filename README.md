# Sales_Dashboard--Excel
Sales Dashboard in Excel

Project Overview

This project presents an interactive Sales Dashboard built using Microsoft Excel. The dashboard provides insights into sales performance across different years, regions, product categories, customer segments, and top-selling products.

Objectives

- Analyze sales performance trends.
- Identify top-performing regions and products.
- Monitor key sales metrics using KPI cards.
- Create a visually appealing dashboard for business decision-making.

Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- KPI Cards
- Data Cleaning and Transformation

Key Performance Indicators (KPIs)

- Total Sales: $2,261,536.78
- Total Orders: 9,800
- Average Sales: $230.77

Dashboard Insights

- West region generated the highest sales.
- Technology is the leading product category.
- Consumer is the largest customer segment.
- Canon imageCLASS 2200 Advanced Copier is the top-selling product.

Visualizations

- Sales by Year
- Sales by Region
- Sales by Product Category
- Sales by Segment
- Top 10 Products by Sales
